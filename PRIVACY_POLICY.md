# Privacy Policy

Azure DevOps Side Preview does not collect, store, or transmit any personal or usage data.

## Data Collection
This extension does not collect any data from users. It only applies CSS changes to Azure DevOps pages to adjust the Markdown editor layout.

## Data Sharing
No data is shared with third parties because no data is collected.

## Remote Code
This extension does not load or execute remote code.

## Changes
If this policy changes, the updated version will be published in this repository.

## Contact
If you have questions, please contact the developer via the repository issues page.
